
# Image Gallery

react version - 18.2.0

vite version - 4.4.5




## Run Locally

Clone the project

```bash
  git clone https://github.com/sajanmoon/ImageGallery
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
